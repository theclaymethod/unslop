# Taboo Phrases Reference

AI-generated text follows predictable patterns. This reference catalogs common tells to eliminate.

---

## Throat-Clearing Openers

These phrases delay getting to the point. Cut them entirely.

| Phrase | Why It's Bad |
|--------|--------------|
| "Here's the thing:" | Delays content |
| "The uncomfortable truth is" | Dramatic, unnecessary |
| "It turns out" | Hedging |
| "The real [X] is" | Implies prior deception |
| "Let me be clear" | Defensive positioning |
| "The truth is," | Implies prior lying |
| "I'll say it again:" | Redundant |
| "I'm going to be honest" | Implies prior dishonesty |
| "Can we talk about" | Permission-seeking |
| "Here's what I find interesting" | Self-referential |
| "Here's the problem though" | Delays content |
| "Let's be real" | Performative |
| "Here's the deal:" | Informal padding |
| "Here's what nobody tells you:" | Clickbait-style |

---

## Emphasis Crutches

These phrases try to manufacture importance. The content should speak for itself.

| Phrase | Why It's Bad |
|--------|--------------|
| "Full stop." | Performative finality |
| "Period." | Same as above |
| "Let that sink in." | Condescending |
| "This matters because" | Tells instead of shows |
| "Make no mistake" | Dramatic positioning |
| "Here's why that matters" | Over-explaining |
| "Read that again." | Condescending |
| "This is important." | Shows insecurity in content |
| "This cannot be overstated." | Overstating the overstating |

| "Spoiler:" | Unnecessary framing |

## Structural Patterns to Avoid

### Binary Contrasts (False Drama)

| Pattern | Fix |
|---------|-----|
| "Not because X. Because Y." | State Y directly |
| "[X] isn't the problem. [Y] is." | "The problem is Y" |
| "It feels like X. It's actually Y." | Just state Y |
| "Not X. But Y." | "Y matters here" |

### Dramatic Fragmentation

| Pattern | Fix |
|---------|-----|
| "[Noun]. That's it. That's the [thing]." | Complete sentences |
| "X. And Y. And Z." | Normal flow |
| "This unlocks something. [Word]." | Trust content |

### Rhetorical Setups

| Pattern | Fix |
|---------|-----|
| "What if [reframe]?" | Make the point directly |
| "Here's what I mean:" | Cut—redundant |
| "Think about it:" | Cut—condescending |
| "And that's okay." | Cut—unnecessary permission |

### Rhythm Problems

| Pattern | Fix |
|---------|-----|
| Three-item lists | Use two or one |
| Questions answered immediately | Let questions breathe or cut |
| Paragraphs starting with "So" | Start with content |
| Every paragraph ends punchily | Vary endings |
| Em-dashes before reveals | Use periods or commas |
| Staccato fragmentation | Don't stack short punchy sentences |

## Quick Reference: Detection Regex

```
# Throat-clearing (case-insensitive)
(?i)(here's the thing|the uncomfortable truth|it turns out|let me be clear|the truth is|let's be real)

# Emphasis crutches
(?i)(full stop\.|period\.|let that sink in|make no mistake|read that again)

# Binary contrast pattern
(?i)(not because .+\. because|isn't the problem\.|feels like .+\. it's actually)

# Fragmentation
\. (That's it\.|And that's|That's the)

# Filler adverbs
(?i)^(interestingly|importantly|crucially|fundamentally),
```
